# 🗂️ Pages 프로젝트 위키

이 저장소는 다양한 웹 도구와 미니 프로젝트를 한 곳에서 관리하는 **프로젝트 위키**입니다.  
각 프로젝트는 독립적으로 동작하며, 실생활에 유용한 계산기, 생성기, 정보 페이지 등을 제공합니다.

---

## 📚 프로젝트 목록

### 🎰 [복권 번호 생성기](./showmethemoney/)
- **로또 6/45**와 **연금복권 720+** 번호를 쉽고 빠르게 생성할 수 있는 웹 도구입니다.
- 원하는 개수만큼 로또 번호 세트, 연금복권 조별 번호를 다양한 방식으로 생성할 수 있습니다.
- [자세히 보기](./showmethemoney/README.md)

### 📊 [연금저축 적립식 배분 계산기](./howtodistribute/)
- 매달 투자할 금액과 종목별 비중을 입력하면, 각 종목에 배분할 금액과 매수 주수를 계산해주는 도구입니다.
- 비중 합계가 100%가 아니어도 자동으로 비례 배분하며, 입력값은 브라우저에 자동 저장됩니다.
- [자세히 보기](./howtodistribute/README.md)

### 🗺️ 세계 여행지도 *(준비 중)*
- 여행한 나라를 표시할 수 있는 세계지도 기능을 준비 중입니다.

### ☔ 오늘 진짜 비옴? *(준비 중)*
- 여러 날씨 정보를 종합해 한눈에 보여주는 날씨 정보 페이지를 준비 중입니다.
